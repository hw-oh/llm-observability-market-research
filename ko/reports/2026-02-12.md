# 주간 LLM Observability 시장 조사 보고서
**날짜**: 2026-02-12 | **모델**: google/gemini-3-pro-preview | **데이터 수집일**: 2026-02-12

## 1. 요약 (Executive Summary)

- MLflow는 시각적 Judge Builder와 MemAlign 최적화 도구를 출시했으며, 멀티 워크스페이스 관리를 위한 Organization Support를 도입했습니다.
- LangSmith는 셀프 호스팅 플랫폼을 v0.13으로 업데이트하고, 새로운 CLI 기능과 Pairwise 비교 큐를 통해 개발자 도구를 확장했습니다.
- Langfuse는 보안 추적을 위한 Organization Audit Log Viewer를 출시하고, 셀프 호스팅 서비스에 PII 마스킹 기능을 추가했습니다.
- Braintrust는 Cursor IDE 지원을 통해 개발자 워크플로우 통합을 심화했으며, Temporal 및 MCP 서버 아키텍처에 대한 Tracing 기능을 추가했습니다.
- Arize Phoenix는 OpenInference 기반 Tracing 플랫폼 내에서 도구 선택 및 호출 정확도를 측정하는 특화된 Eval 도구를 도입했습니다.
- W&B Weave는 멀티모달 Eval을 위한 고급 오디오 모니터와 동적 리더보드를 배포했으며, 전용 및 고객 관리형 배포 옵션을 지원합니다.

**AI 기반 시장 인사이트**:

> MLflow가 시각적 Judge Builder와 MemAlign 최적화 도구를 도입함에 따라, 프로그래밍 방식의 Eval 및 리더보드 시각화 분야에서 Weave가 가졌던 차별화 요소에 도전하고 있습니다.
> Langfuse의 전용 Audit Log Viewer 및 PII 마스킹 출시는 명시적인 컴플라이언스 가시성 측면에서 Weave의 현재 엔터프라이즈 기능 세트에 존재하는 공백을 부각시킵니다.
> Braintrust가 Cursor IDE 및 Temporal 워크플로우와 직접 통합됨으로써, 복잡한 에이전트 디버깅을 위한 주요 개발자 중심 플랫폼으로서의 Weave의 입지를 위협하고 있습니다.

## 3. 신규 기능 (최근 30일)

### [W&B Weave](https://app.getbeamer.com/wandb/en)
- **Audio Monitors**: LLM Judge를 사용하여 텍스트와 함께 오디오 출력(MP3/WAV)을 관찰하고 판정하는 모니터링 기능. (2026-02-01, Evaluation Integration)
- **Dynamic Leaderboards**: 지속적인 커스터마이징 및 CSV 내보내기 기능이 포함된 Eval 결과 기반 자동 생성 리더보드. (2026-01-29, Evaluation Integration)
- **Playground 내 커스텀 LoRA**: Weave Playground에서 직접 Fine-tuning된 커스텀 LoRA 가중치를 로드하고 테스트하는 기능. (2026-01-16, Experiment / Improvement Loop)

### [LangSmith](https://changelog.langchain.com)
- **Tracing 미리보기 커스터마이징**: UI에서 Tracing이 미리 보여지는 방식을 사용자 정의하는 기능. (2026-02-06, Core Observability)
- **Google Gen AI Wrapper**: Google Gen AI 모델을 위한 새로운 Wrapper 지원. (2026-01-31, DevEx / Integration)
- **LangSmith Self-Hosted v0.13**: 엔터프라이즈 배포를 위한 업데이트된 셀프 호스팅 버전. (2026-01-16, Enterprise & Security)

### [Langfuse](https://langfuse.com/changelog)
- **버전 관리된 데이터셋에서 실험 실행**: 특정 버전 타임스탬프의 데이터셋을 가져와 재현성을 위해 과거 버전에서 실험을 실행하는 기능. (2026-02-11, Experiment / Improvement Loop)
- **Org Audit Log Viewer**: 보안 및 액세스 이벤트를 추적하기 위한 조직 수준의 감사 로그 뷰어. (2026-02-09, Enterprise & Security)
- **사고/추론 과정 렌더링 (Render Thinking/Reasoning)**: Tracing 상세 보기에서 LLM 응답의 사고 및 추론 부분을 별도로 렌더링. (2026-02-09, Core Observability)
- **단일 관찰 Eval (Single Observation Evals)**: 단일 관찰 데이터에 대해 직접 Eval을 실행하는 기능 지원. (2026-02-09, Evaluation Integration)
- **Tracing을 위한 수정된 출력 (Corrected Outputs)**: Fine-tuning 데이터셋 구축을 위해 Tracing 뷰에서 직접 개선된 LLM 출력 버전을 캡처. (2026-01-14, Experiment / Improvement Loop)

### [Braintrust](https://braintrust.dev/docs/changelog)
- **커스텀 뷰 내 첨부 파일 렌더링**: 커스텀 Tracing 뷰에서 이미지, 비디오, 오디오를 직접 렌더링 지원. (2026-02-01, Core Observability)
- **Tracing 기원 탐색**: 로그의 Tracing에서 해당 데이터가 생성된 프롬프트나 데이터셋 행으로 역추적. (2026-02-01, Core Observability)
- **Tracing 레벨 Scoring 도구**: 커스텀 코드 Scoring 도구가 다단계 Eval을 위해 전체 실행 Tracing에 접근 가능. (2026-02-01, Evaluation Integration)
- **LangSmith 통합**: Tracing 및 Eval 호출을 LangSmith와 Braintrust 양쪽으로 전송하는 Wrapper. (2026-02-01, DevEx / Integration)
- **Cursor 통합**: Cursor 에디터 내에서 Braintrust MCP 서버를 자동으로 구성하는 확장 프로그램. (2026-02-01, DevEx / Integration)
- **이미지 렌더링 보안 제어**: 로그 내 외부 이미지에 대해 구성 가능한 모드(자동 로드, 클릭 시 로드, 차단) 제공. (2026-02-01, Enterprise & Security)
- **집계 기능이 포함된 단일 Span 필터**: 집계된 Tracing 분석을 위해 단일 Span 필터와 GROUP BY 결합 지원. (2026-02-01, Monitoring & Metrics)
- **Python, Ruby, Go 자동 Instrumentation**: 주요 언어에 대한 코드 수정 없는 Tracing 지원. (2026-01-29, DevEx / Integration)
- **Temporal 통합**: Temporal 워크플로우 및 액티비티의 자동 Tracing. (2026-01-21, DevEx / Integration)
- **리뷰를 위한 칸반 레이아웃**: 플래그가 지정된 Span 및 리뷰 관리를 위한 드래그 앤 드롭 인터페이스. (2026-01-21, Evaluation Integration)

### [MLflow](https://mlflow.org/releases)
- **Organization Support**: 실험 및 리소스를 정리하기 위한 멀티 워크스페이스 환경 지원. (2026-02-12, Enterprise & Security)

- **구성 가능한 이메일 추출**: JMESPath를 사용하여 OAuth2 프로바이더로부터 커스텀 이메일 추출 지원. (2026-01-28, Enterprise & Security)

- **Tracing과 함께 어노테이션 내보내기**: 수동 레이블 및 Eval 점수를 포함한 Tracing 내보내기 CLI 지원. (2026-01-19, DevEx / Integration)
- **AI 어시스턴트용 CLI 터미널 액세스**: AI 코딩 어시스턴트(Cursor, Windsurf)가 Phoenix 데이터를 쿼리할 수 있도록 설계된 CLI 기능. (2026-01-17, DevEx / Integration)

## 5. 엔터프라이즈 신호

- Langfuse는 보안 및 액세스 이벤트를 추적하기 위해 Organization Audit Log Viewer를 출시했습니다.
- MLflow는 엔터프라이즈 팀을 위한 멀티 워크스페이스 환경 관리를 위해 Organization Support를 도입했습니다.
- LangSmith는 셀프 호스팅 버전(v0.13)을 업데이트했으며 SOC 2/HIPAA 컴플라이언스를 유지하고 있습니다.
- Braintrust는 이미지 렌더링 보안 제어 기능을 추가했으며 온프레미스 배포를 지원합니다.
- W&B Weave는 데이터 주권 보장을 위해 전용 및 고객 관리형 배포 옵션을 제공합니다.
