# 주간 LLM Observability 시장 조사 보고서
**날짜**: 2026-02-12 | **모델**: google/gemini-3-pro-preview | **데이터 수집일**: 2026-02-12

## 1. 요약 (Executive Summary)

- LangSmith는 셀프 호스팅 플랫폼 v0.13을 출시하여 온프레미스 역량을 강화하고, 다단계 추론 시각화를 위한 LangGraph 연동을 심화했습니다.
- MLflow는 멀티 워크스페이스 관리를 위한 Organization Support와 자동화된 평가 도구 생성을 간소화하는 시각적 Judge Builder를 도입했습니다.
- W&B Weave는 동적 리더보드와 커스텀 Scoring 기능을 출시했으며, 멀티모달 Eval 워크플로우를 위해 설계된 독특한 오디오 모니터를 선보였습니다.
- Braintrust는 구성 가능한 이미지 렌더링 보안 제어 기능을 구현하고, 도구 호출 Tracing을 위한 새로운 Temporal 연동을 통해 에이전트 Observability를 확장했습니다.
- Langfuse는 컴플라이언스 가시성을 높이기 위해 Organization Audit Log Viewer를 출시하고, 복잡한 체인의 지연 시간을 디버깅하기 위한 타임라인 뷰를 추가했습니다.
- Arize Phoenix는 도구 선택 정확도에 특화된 메트릭을 출시하며 에이전트 Eval에 집중했으며, 셀프 호스팅 배포를 위한 RBAC 지원을 확정했습니다.

**AI 시장 통찰**:

> LangSmith와 Arize Phoenix는 특정 도구 선택 평가 도구와 추론 Tracing을 통해 에이전트 워크플로우를 공격적으로 공략하고 있으며, 이는 Weave가 기본적인 도구 호출 Tracing을 넘어 에이전트 시각화를 확장하도록 압박하고 있습니다.
> Weave는 강력한 온프레미스 옵션을 제공하지만, Langfuse의 새로운 Audit Log Viewer와 LangSmith의 v0.13 셀프 호스팅 출시는 PII 마스킹 및 컴플라이언스 로그와 관련된 Weave의 명시적 문서화 공백을 부각시킵니다.
> Braintrust의 깊은 IDE 연동과 MLflow의 시각적 Judge Builder는 코딩 환경과 Eval 로직 간의 더 긴밀한 결합을 제공함으로써 Weave의 개발자 우선 포지셔닝에 도전하고 있습니다.

## 3. 신규 기능 (최근 30일)

### [W&B Weave](https://app.getbeamer.com/wandb/en)
- **오디오 모니터**: LLM Judge를 사용하여 텍스트와 함께 오디오 출력을 관찰하고 판별하는 모니터입니다. (2026-02-01, Monitoring & Metrics)
- **동적 리더보드**: 영구적인 커스터마이징 및 CSV 내보내기 기능이 포함된 Eval 기반 자동 생성 리더보드입니다. (2026-01-29, Evaluation Integration)
- **Playground 내 커스텀 LoRA**: Weave Playground에서 직접 커스텀 Fine-tuning된 LoRA 가중치를 테스트하고 평가하는 기능입니다. (2026-01-16, Experiment / Improvement Loop)

### [LangSmith](https://changelog.langchain.com)
- **Python SDK v0.7.1**: LangSmith 플랫폼 연결을 위한 클라이언트 라이브러리 업데이트입니다. (2026-02-10, DevEx / Integration)
- **Trace 미리보기 커스텀**: UI에서 Trace가 미리 보여지는 방식을 커스터마이징하는 기능입니다. (2026-02-06, Core Observability)
- **Google Gen AI Wrapper 내보내기**: SDK에서 Google Gen AI 래퍼를 위한 내보내기 기능입니다. (2026-01-31, DevEx / Integration)
- **LangSmith 셀프 호스팅 v0.13**: 셀프 호스팅 플랫폼의 새로운 릴리스 버전입니다. (2026-01-16, Enterprise & Security)

### [Langfuse](https://langfuse.com/changelog)
- **Org Audit Log Viewer**: 조직 Audit Log를 확인하기 위한 새로운 UI입니다. (2026-02-09, Enterprise & Security)
- **Thinking/Reasoning 파트 렌더링**: Trace 상세 정보에서 추론 모델 등의 생각(Thinking) 및 추론(Reasoning) Trace 렌더링을 지원합니다. (2026-02-09, Core Observability)
- **단일 Observation Eval**: 개별 Observation에 평가를 추가하는 기능입니다. (2026-02-09, Evaluation Integration)
- **이벤트 기반 Trace 테이블**: Observation/Trace 테이블을 위한 새로운 이벤트 기반 뷰 모드입니다. (2026-02-09, Core Observability)
- **Trace용 교정된 출력(Corrected Outputs)**: Fine-tuning 데이터셋 구축을 위해 Trace 뷰에서 직접 개선된 LLM 출력 버전을 캡처합니다. (2026-01-14, Experiment / Improvement Loop)

### [Braintrust](https://braintrust.dev/docs/changelog)
- **커스텀 뷰 내 첨부 파일 렌더링**: 커스텀 Trace 뷰에서 이미지, 비디오, 오디오 및 기타 첨부 파일을 직접 렌더링할 수 있도록 지원합니다. (2026-02, Core Observability)
- **Trace 기원 탐색**: 로그의 Trace에서 해당 Trace가 시작된 프롬프트나 데이터셋 행으로 역탐색합니다. (2026-02, Experiment / Improvement Loop)
- **Trace 레벨 Scoring**: 커스텀 코드 Scoring 도구가 다단계 워크플로우 평가를 위해 전체 실행 Trace에 접근할 수 있습니다. (2026-02, Evaluation Integration)
- **LangSmith 연동**: Tracing 및 Eval 호출을 LangSmith와 Braintrust 양쪽으로 보내거나 Braintrust로만 라우팅하는 래퍼입니다. (2026-02, DevEx / Integration)
- **Cursor 연동**: Cursor에서 로그 및 실험을 쿼리하기 위해 Braintrust MCP 서버를 자동으로 구성하는 확장 프로그램입니다. (2026-02, DevEx / Integration)
- **이미지 렌더링 보안 제어**: 이미지 URL을 통한 민감한 데이터 유출을 방지하기 위해 구성 가능한 모드(자동 로드, 클릭 시 로드, 차단)를 제공합니다. (2026-02, Enterprise & Security)
- **집계 기능이 포함된 단일 스팬 필터**: 단일 스팬 필터를 GROUP BY와 결합하여 스팬 레벨 조건에 따라 Trace를 집계합니다. (2026-02, Monitoring & Metrics)
- **자동 Instrumentation (Python, Ruby, Go)**: Python, Ruby, Go 애플리케이션을 위한 코드 수정 없는 Tracing 지원입니다. (2026-01, DevEx / Integration)
- **Temporal 연동**: 분산 Tracing 지원을 통해 Temporal 워크플로우 및 액티비티를 자동으로 추적합니다. (2026-01, DevEx / Integration)
- **리뷰용 칸반 레이아웃**: 플래그가 지정된 스팬 및 리뷰 상태 관리를 위한 드래그 앤 드롭 인터페이스입니다. (2026-01, Evaluation Integration)
