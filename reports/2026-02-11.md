# W&B Weave — 주간 경쟁사 인텔리전스 리포트
**날짜**: 2026-02-11 | **모델**: google/gemini-3-pro-preview | **데이터 수집일**: 2026-02-11

## 1. Executive Summary

- Humanloop의 서비스 종료(Sunset) 발표로 인해 기존 엔터프라이즈 고객들의 대규모 이탈이 시작되었으며, 이를 흡수하기 위한 경쟁사들의 마이그레이션 지원 경쟁이 치열해질 전망입니다.
- 단순 트레이싱을 넘어 '에이전트 도구 선택(Tool Selection)' 정확도와 '사고 과정(Thinking Process)'을 시각화하고 평가하는 기능이 LangSmith, Arize, Langfuse를 중심으로 표준화되고 있습니다.
- Braintrust는 Cursor, Claude Code 등 AI 코딩 도구 및 다양한 언어(Java, Go 등) SDK 확장을 통해 개발자 워크플로우(DevEx)의 운영체제로 자리 잡으려 시도하고 있습니다.
- 데이터 보안이 중요한 엔터프라이즈 시장에서 Langfuse, Arize Phoenix, Helicone의 '완전한 자체 호스팅(Self-hosting)' 옵션이 강력한 세일즈 포인트로 작용하고 있습니다.
- Logfire(SQL 기반)와 Helicone(게이트웨이 기반)은 평가보다는 운영 모니터링과 비용/인프라 제어에 특화된 틈새시장을 공략하며 차별화하고 있습니다.

> **한줄 총평**: Humanloop의 종료로 인한 시장 재편 기회 속에서, 에이전트 워크플로우 심층 분석과 AI 코딩 도구 통합이 2026년 상반기 경쟁 우위의 핵심 키워드로 부상했습니다.

## 3. New Features This Week

### LangSmith
- **LangSmith Fetch (CLI)**: 터미널에서 직접 LangSmith 트레이스를 조회하고 디버깅할 수 있는 CLI 도구 (2025-12-10, DevEx / Integration)
- **Pairwise Annotation Queues**: 두 에이전트의 출력을 나란히 비교하여 승자를 선택하는 방식의 구조화된 평가 큐 (2025-12-17, Evaluation Integration)
- **Unified Cost Tracking**: LLM, 도구, 검색 등을 포함한 에이전트 애플리케이션 전체 스택에 대한 통합 비용 추적 (2025-12-02, Monitoring & Metrics)
- **Polly (AI Assistant Beta)**: LangSmith 내에서 에이전트 디버깅과 분석을 돕는 AI 어시스턴트 (2025-12-10, DevEx / Integration)

### Arize Phoenix
- **Claude Opus 지원 (Playground)**: 프롬프트 플레이그라운드에 Claude Opus 4.6 모델 지원 추가 (2026-02-09, Experiment / Improvement Loop)
- **도구 선택(Tool Selection) Evaluator**: 에이전트가 적절한 도구를 선택했는지 평가하는 새로운 메트릭 추가 (2026-02-06, Agent / RAG Observability)
- **Faithfulness Evaluator**: 기존 Hallucination 평가를 대체/개선한 신뢰성(Faithfulness) 평가 모듈 도입 (2026-02-02, Evaluation Integration)
- **커스텀 메트릭 커서 규칙**: LLM 분류 평가를 기반으로 새로운 내장 메트릭을 생성하는 규칙 엔진 추가 (2026-01-21, Monitoring & Metrics)

### Braintrust
- **Trace-level Scorers**: 단일 스팬이 아닌 전체 트레이스 실행 흐름을 분석하여 다단계 워크플로우를 평가하는 기능 (2026-02, Evaluation Integration)
- **LangSmith Integration**: LangSmith와 Braintrust로 트레이스를 동시에 전송하거나 라우팅할 수 있는 래퍼 제공 (2026-02, DevEx / Integration)
- **Auto-instrumentation (Python, Ruby, Go)**: 코드 변경 없이 주요 라이브러리를 자동으로 추적하는 기능 추가 (2026-01, DevEx / Integration)
- **Temporal Integration**: Temporal 워크플로우 및 액티비티를 자동으로 추적하여 분산 시스템 가시성 확보 (2026-01, DevEx / Integration)
- **Claude Code Integration**: Anthropic의 에이전트 코딩 도구인 Claude Code와의 통합 지원 (2025-12, DevEx / Integration)

### Langfuse
- **Corrected Outputs (수정된 출력)**: 트레이스 뷰에서 LLM의 출력을 직접 수정하고 저장하여 파인튜닝 데이터셋으로 활용 가능 (2026-01, Experiment / Improvement Loop)
- **Inline Comments (인라인 코멘트)**: 트레이스 내 특정 텍스트 구간에 코멘트를 달아 협업 및 피드백 강화 (2026-01, Evaluation Integration)
- **Hosted MCP Server**: Model Context Protocol 서버를 내장하여 에이전트가 Langfuse의 프롬프트를 직접 가져오고 업데이트 가능 (2025-11, DevEx / Integration)
- **Dataset Item Versioning**: 데이터셋 아이템의 생성, 수정, 삭제 등 모든 변경 사항을 자동으로 버전 관리 (2025-12, Experiment / Improvement Loop)

- **SQL 기반 분석**: 수집된 모든 트레이스 및 로그 데이터를 SQL로 자유롭게 쿼리하고 시각화하는 기능 (2024-05, Monitoring & Metrics)

## 6. Enterprise Signals

- Humanloop의 Anthropic 인수로 인한 서비스 종료는 단일 벤더 종속성(Lock-in)에 대한 기업들의 우려를 자극하여 오픈소스 또는 자체 호스팅 가능한 대안(Langfuse, Arize) 선호도를 높이고 있습니다.
- Braintrust와 Helicone은 AI Proxy(Gateway) 기능을 통해 모델 접근 제어, 속도 제한, 통합 과금 관리 등 IT 거버넌스 요구사항을 충족시키며 엔터프라이즈 침투율을 높이고 있습니다.
- LangSmith와 Langfuse는 'Annotation Queue'와 같은 휴먼 피드백 워크플로우를 강화하여, 개발자뿐만 아니라 도메인 전문가가 참여하는 엔터프라이즈급 품질 관리 프로세스를 지원합니다.
- Logfire의 SQL 기반 분석과 Langfuse의 ClickHouse 도입은 대규모 트래픽을 처리해야 하는 엔터프라이즈 환경에서의 데이터 조회 성능과 분석 유연성을 강조하는 신호입니다.

## 7. Insights

### Humanloop 이탈 고객 확보전 (The Great Migration)

Humanloop의 서비스 종료(2025년 9월)는 Weave에게 큰 기회입니다. 특히 Humanloop의 강점이었던 '프롬프트 관리'와 '비개발자 친화적 UI'를 선호하던 고객층을 흡수하기 위해, Weave의 Board 기능과 프롬프트 관리 기능을 강조한 마이그레이션 가이드 배포가 시급합니다.

### 에이전트 품질 평가의 고도화

단순한 실행 추적을 넘어, Arize의 'Tool Selection Evaluator'나 Langfuse의 'Thinking Process' 시각화처럼 에이전트의 의사결정 품질을 정량화하려는 시도가 늘고 있습니다. Weave 또한 에이전트의 내부 로직(CoT)을 평가하는 전용 메트릭과 뷰를 강화해야 합니다.

### 개발 도구(IDE)와의 경계 붕괴

Braintrust가 Cursor 및 Claude Code와 통합하고, Logfire가 Pytest 플러그인을 제공하는 등 옵저버빌리티 도구가 개발 환경(IDE/CLI) 깊숙이 침투하고 있습니다. 이는 개발자가 코드를 작성하는 시점부터 자연스럽게 플랫폼을 사용하도록 유도하는 강력한 락인(Lock-in) 전략입니다.

## 8. Watchlist

- Humanloop 기존 고객을 타겟팅하는 경쟁사(Langfuse, LangSmith)의 전용 마이그레이션 도구 출시 여부
- Braintrust의 AI 코딩 에이전트(Claude Code 등) 통합이 실제 개발자 점유율 확대에 미치는 영향
- LangSmith의 'Polly'와 같은 AI 어시스턴트 기능이 디버깅 경험을 얼마나 자동화할 수 있는지 여부
- Arize Phoenix가 주도하는 OpenInference 표준의 에이전트 평가 메트릭(Tool Selection 등) 확산 속도
