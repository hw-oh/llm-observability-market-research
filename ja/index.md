## 最新レポート

<!-- LATEST_REPORT_START -->

[📋 最新レポート (2026-02-11)](./reports/2026-02-11.md)


- 市場集約の機会: Anthropicによる Humanloop の買収と、それに伴うサービス終了（2025年9月）は、プロンプトエンジニアリングや評価ツールの移行先を探しているエンタープライズ顧客を獲得するための、緊急かつ優先度の高い機会を生み出しています。
- 「Agent Ops」へのシフト: 競合他社は、単純なトレーシングを超えて、フルライフサイクル管理へと積極的に移行しています。LangSmith による「Agent Servers」と「Unified Cost Tracking（統合コスト追跡）」のリリースは、ホスティングと財務運用へのピボットを象徴しており、Weave に対して本番環境へのデプロイメントストーリーを定義するよう迫っています。
- 可視化の軍拡競争: 循環型/グラフベースのエージェントワークフローに特化した可視化への明確なトレンドがあります。LangSmith (LangGraph)、Langfuse (Agent Graphs)、Arize Phoenix は専用のグラフビューで差別化を図っており、複雑なエージェントに対して線形なトレースリストは時代遅れになりつつあります。
- OpenTelemetry と MCP による標準化: Arize Phoenix、Langfuse、Logfire は、標準規格として OpenTelemetry (OTel) と Model Context Protocol (MCP) に集約されつつあります。この圧力により、独自の SDK に対する主要なセールスポイントとして、相互運用性と「ベンダーロックインなし」が強調されています。
- 評価機能の成熟: 市場は「マネージド・エバリュエーター」と「アノテーション・キュー」へと移行しました。LangSmith と Langfuse は現在、ヒューマン・イン・ザ・ループ（人間によるレビュー）のための専用ワークフローを提供しており、評価を単なる開発者向けのユーティリティではなく、チームスポーツとして位置づけています。
- 多言語展開: Braintrust は、Java、Go、Ruby、C# へのネイティブ SDK サポートを拡大することで、エンタープライズのバックエンド市場を積極的にターゲットにしており、多くのオブザーバビリティツールにおける Python/JS 優位の状況に挑戦しています。
- 差別化要因としての SQL: Logfire と Braintrust は、トレースに対する SQL ベースのクエリ機能をパワーユーザーに提供し、構築済みのダッシュボードを超えた、きめ細かくアドホックなデータ分析機能を求めるエンジニアのニーズに応えています。

> Weave は「トレーニングから本番まで」の系統管理においてリードを保っていますが、エージェントのデプロイに関しては LangSmith から、オープン標準とコストの可視化に関しては Langfuse や Arize Phoenix からの激しいプレッシャーに直面しています。
