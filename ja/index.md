## エグゼクティブサマリー

<!-- LATEST_REPORT_START -->

[📋 最新レポート (2026-02-13)](./reports/2026-02-13.md)


- Langfuse は、推論ステップを可視化するための「thinking」トレースのレンダリングサポートを導入しました。一方、Arize Phoenix は、エージェントワークフローにおけるツール選択と忠実性（faithfulness）を評価するための専用メトリクスをリリースしました。
- MLflow はバージョン 3.10 をローンチし、大規模なエンタープライズ環境内でのマルチワークスペース管理とアクセス制御を可能にする Organization Support を導入しました。
- LangSmith は、Streaming サポートとエージェント実行グラフの可視化機能を維持しつつ、オンプレミスデプロイメントのインフラ安定性を向上させるため、Self-Hosted v0.13 をリリースしました。
- Braintrust は、既存の「Loop」AI エバリュエーターアシスタントに加え、ヒューマンインザループによる品質管理ワークフローを構造化するための専用の「Review」スパンタイプを追加しました。
- W&B Weave は、VPC オプションにおいて SOC 2 Type II および HIPAA コンプライアンスを確認し、ネイティブな Model Context Protocol (MCP) 統合によってエージェントの Observability 機能を差別化しました。

> **マーケットインサイト**: Arize Phoenix による特化型エージェントメトリクスのリリースと Langfuse の「thinking」トレースサポートは、Weave のエージェント Observability の深みに直接的な挑戦を突きつけています。技術的な差別化を維持するためには、Weave 独自の Model Context Protocol (MCP) 統合を引き続き活用していく必要があります。
