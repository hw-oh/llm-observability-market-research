## エグゼクティブサマリー

<!-- LATEST_REPORT_START -->

[📋 最新レポート (2026-02-13)](./reports/2026-02-13.md)


- LangSmith は Self-Hosted v0.13 をリリースし、LLM、ツール、検索コンポーネントを網羅する統合的なコストトラッキング機能を導入しました。
- Langfuse はコンプライアンス監視用の Organization Audit Log Viewer をローンチし、エージェントの可視性を向上させるために推論ステップ（reasoning step）のレンダリング機能を追加しました。
- MLflow は Tracking Server 内でマルチワークスペース管理を可能にする Organization Support を導入し、ガバナンス機能を専門的な LLM ツールに近い水準まで引き上げました。
- Braintrust は、エンタープライズにおける再現可能な実験と回帰テストをサポートするため、Eval 用のきめ細かなキャッシュコントロール機能を追加しました。
- Arize Phoenix は、ツールの選択と忠実性（faithfulness）に特化した新しいエバリュエーターをリリースし、あわせて LLM judge を設定するための GUI ベースのウィザードを公開しました。
- W&B Weave は、マルチモーダル入力の Tracing へのネイティブサポートを維持しつつ、チーム間での Eval 比較を容易にするダイナミックリーダーボードをローンチしました。

> **マーケットインサイト**: Langfuse と MLflow が監査ログと組織レベルの管理機能を同時にリリースしたことは、管理ガバナンスの面で差を埋めるものであり、Weave のエンタープライズ対応という優位性に挑戦しています。 | Arize Phoenix が LLM judge 用の GUI ベースのウィザードを導入したことは、Weave の既存のノーコード Eval 作成ワークフローと直接競合し、judge 設定における使いやすさの差別化を汎用化させています。 | LangSmith の高度なエージェントサンドボックスや Langfuse の推論ステップレンダリングは、複雑なエージェントワークフローに対する高レベルな実行グラフが不足している Weave のエージェント Observability における具体的な課題を浮き彫りにしています。
