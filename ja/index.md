# 競合インテリジェンス・ボット

[詳細比較](./comparison) · [製品詳細](./competitor-detail) · [競合インテリジェンス (内部用)](https://docs.google.com/presentation/d/125NLww3icyIEa8qq0668gVTEcQuuF9RjAcSo0B3Xzqo/edit)

## 最新レポート

<!-- LATEST_REPORT_START -->

[📋 最新レポート (2026-02-11)](./reports/2026-02-11.md)


- LangSmith は「LangGraph Cloud」によって、受動的なオブザーバビリティから能動的なランタイム管理へと積極的に移行しており、Weave がデータを取得する前にエージェントのライフサイクル全体を囲い込もうとしています。
- Braintrust は、IDE (Cursor) への直接統合や、Weave が現在対応していないバックエンド言語 (Java, Go, C#) のサポートにより、「エンタープライズ・エンジニアリング」の側面から効果的に側面攻撃を仕掛けています。
- Humanloop の買収とサービス終了は、行き場を失った顧客を獲得する即時的な戦術的機会を生み出していますが、Langfuse の優れた Prompt CMS は、この移行先として強力な候補となっています。
- Arize Phoenix と Langfuse は、強力な「ローカルファースト」およびセルフホストのストーリーでボトムアップの採用プロセスを圧迫しており、SaaS の導入を躊躇するセキュリティ意識の高いエンジニアに訴求しています。
- Logfire は Pydantic との深い統合により、「フルスタック」トレース (データベース + API + LLM) を独占しており、LLM 以外のインフラコンポーネントに対する Weave の可視性の欠如を露呈させています。
- エージェントのオブザーバビリティが主要な戦場になりつつあります。LangSmith と Langfuse は、循環ワークフローにおいて Weave の標準的なトレースビューを凌駕する、専用のグラフ/推論ビジュアライゼーションをリリースしました。

> Weave はモデルのトレーニングから本番環境へのリネージにおいて優位性を保っていますが、エージェントのオーケストレーションでは LangSmith から、エンタープライズ開発者のワークフローでは Braintrust からの激しい圧力に直面しています。

## 新機能 (直近30日間)

### LangSmith

- **トレースプレビューのカスタマイズ**: トレースの入力および出力のどの部分をトレーステーブルに直接表示するかをユーザーが設定できるようになり、カスタムデータ構造の使い勝手が向上しました。(2026-02-06)
- **LangSmith セルフホスト版 v0.13**: セルフホスト顧客向けのメジャーリリース。「Insights」機能の追加やパフォーマンス改善など、クラウド版との機能パリティ（同等性）を拡大。(2026-01-16)
- **SDK v0.7.1**: Python/JS クライアントライブラリのアップデート。最新の LangChain バージョンとの互換性確保と安定性の向上。(2026-02-10)

### Arize Phoenix

- **Claude Opus 4.6 サポート**: Playground に Claude Opus 4.6 のサポートを追加し、並列比較が可能に。(2026-02-09)
- **ツール選択エバリュエーター**: エージェントのツール選択ロジックの精度を測定するために特別に設計された新しいエバリュエーター。(2026-02-06)
- **FaithfulnessEvaluator**: よりきめ細かな RAG チェックのため、新しい FaithfulnessEvaluator を導入 (HallucinationEvaluator は非推奨に)。(2026-02-02)
- **ツール呼び出し精度メトリクス**: エージェントが正しくツールを呼び出した頻度を追跡するための特定のメトリクスを追加。(2026-01-27)

### Braintrust

- **トレースの起点へのナビゲート**: トレースを、それを生成した特定のプロンプトバージョンやデータセットの行にリンクさせる機能。(2026-02)
- **トレースレベルのスコーラー**: カスタムスコーラーが実行トレース全体にアクセスできるようになり、マルチステップのワークフローやエージェントの動作を評価可能に。(2026-02)
- **LangSmith 統合**: LangSmith のトレースおよび評価コールを、並行して、あるいは排他的に Braintrust へルーティングするラッパー。(2026-02)
- **Cursor 統合**: 自然言語を介してログの照会や実験結果の取得を行うための、Cursor エディタ用 Braintrust 拡張機能。(2026-02)
- **自動インストルメンテーション (Python/Ruby/Go)**: 主要言語のコードレス・トレースをサポートし、初期セットアップを簡素化。(2026-01)

### Langfuse

- **トレースの修正済み出力**: ファインチューニング用データセットを構築するために、LLM 出力の改善版をトレースビューで直接キャプチャ。(2026-01-14)
- **オブザベーション I/O へのインラインコメント**: トレースの入力および出力内の特定のテキスト選択箇所にコメントを固定。(2026-01-07)

### Logfire

- **マルチトークン・プロジェクト移行**: 複数のトークンを使用しているプロジェクトの移行をサポート。(2026-02-04)
- **OTel Gen AI セマンティックコンベンション**: OpenTelemetry Gen AI セマンティックコンベンションのスカラー属性のサポートを追加。(2026-01-28)
- **Pytest 統合**: pytest 実行をトレースするためのネイティブ統合。(2026-01-26)
- **DSPy 統合**: DSPy フレームワークのインストルメンテーション・サポートを追加。(2026-01-16)
- **Claude SDK インストルメンテーション**: Anthropic Claude SDK 専用のインストルメンテーションを追加。(2026-01-12)
