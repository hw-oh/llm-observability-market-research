# 週刊 LLM Observability 市場調査レポート
**日付**: 2026-02-12 | **モデル**: google/gemini-3-pro-preview | **データ収集日**: 2026-02-12

## 1. エグゼクティブ・サマリー

- Langfuse は、エンタープライズ導入におけるセキュリティとアクセスイベントを追跡するための「組織監査ログビューア（Organization Audit Log Viewer）」をリリースしました。
- MLflow は、マルチワークスペース環境とリソースの管理を可能にする「組織サポート（Organization Support）」を導入しました。
- LangSmith は、オンプレミス・インフラストラクチャの要件に対応するため、セルフホスト版を v0.13 にアップデートしました。
- Braintrust は、設定可能な画像レンダリング・セキュリティ・コントロールを実装し、IDE に直接統合するための Cursor 拡張機能を導入しました。
- Arize Phoenix は、OAuth2 プロバイダー向けの設定可能なメール抽出サポートを追加し、CLI ツールの機能を拡張しました。

**AI による市場インサイト**:

> Langfuse による監査ログと MLflow によるマルチワークスペース・サポートの同時リリースは、エンタープライズのコンプライアンスと階層構造の要件を満たす方向への競争的な転換を示しています。
> Braintrust と Arize Phoenix は、実行後のダッシュボードだけに頼るのではなく、Cursor や CLI などの開発環境にトレースを直接埋め込むことで、オブザーバビリティを開発工程の上流へと移動させています。
> LangSmith、Langfuse、W&B Weave 全体で推論ステップのディープトレースが標準化されたことは、エージェント型ワークフローの可視化が差別化要因ではなく、もはや基本要件になったことを示唆しています。

## 3. 新機能 (直近30日間)

### [LangSmith](https://changelog.langchain.com)

- **トレースプレビューのカスタマイズ**: UI でのトレースプレビューの表示方法をカスタマイズする機能。(2026-02-06, コア・オブザーバビリティ)
- **Python SDK v0.6.9**: pre-commit フックの追加とサンドボックスの非同期エンドポイントの修正。(2026-02-05, DevEx / 統合)
- **Python SDK v0.6.8**: 非 OTEL の Google ADK ラッパーと、オプションの chat/completions 名前パラメータ。(2026-02-02, DevEx / 統合)
- **Python SDK v0.6.7**: Google Gen AI ラッパーのエクスポート。(2026-01-31, DevEx / 統合)
- **JS SDK v0.6.6**: JS SDK 0.4.10 のリリースとサンドボックスのデフォルトエンドポイントの修正。(2026-01-27, DevEx / 統合)
- **LangSmith セルフホスト版 v0.13**: プラットフォームのセルフホスト版のアップデートリリース。(2026-01-16, エンタープライズ & セキュリティ)

### [Langfuse](https://langfuse.com/changelog)
- **単一オブザベーション評価**: 単一のオブザベーションに対して直接評価を実行する機能。(2026-02-01, 評価統合)
- **組織監査ログビューア**: セキュリティとアクセスイベントを追跡するための組織レベルの新しい監査ログビューア。(2026-01-28, エンタープライズ & セキュリティ)
- **推論/思考トレースのレンダリング**: トレース詳細におけるモデル出力の「思考」または推論部分の可視化サポート。(2026-01-28, コア・オブザーバビリティ)
- **修正済み出力 (Corrected Outputs)**: データセット作成のために、トレースビューで LLM 出力の改善版を直接キャプチャ。(2026-01-14, 実験 / 改善ループ)
- **イベントベースのトレーステーブル**: 新しいイベントベースのオブザベーションおよびトレーステーブルビュー。(2026-01-29, コア・オブザーバビリティ)

### [Braintrust](https://braintrust.dev/docs/changelog)
- **トレースレベル・スコアラー**: カスタムコードスコアラーが実行トレース全体にアクセスし、マルチステップのワークフローやエージェントの動作を評価可能に。(2026-02-01, 評価統合)
- **LangSmith 統合**: トレースおよび評価の呼び出しを LangSmith と Braintrust の両方に送信、または Braintrust のみにルーティングするラッパー。(2026-02-01, DevEx / 統合)
- **Cursor 統合**: Cursor からログの照会や実験の実行を行うための Braintrust MCP サーバーを自動構成する拡張機能。(2026-02-01, DevEx / 統合)
- **画像レンダリング・セキュリティ・コントロール**: 画像 URL からの機密データ漏洩を防ぐための設定可能なモード（自動読み込み、クリックして読み込み、ブロック）。(2026-02-01, エンタープライズ & セキュリティ)
- **集計付きシングルスパンフィルタ**: シングルスパンフィルタと GROUP BY を組み合わせ、スパンレベルの条件に基づいてトレースを集計。(2026-02-01, モニタリング & メトリクス)
- **自動インストルメンテーション (Python, Ruby, Go)**: Python、Ruby、Go アプリケーション向けのコード不要のトレースサポート。(2026-01-20, DevEx / 統合)
- **Temporal 統合**: 親子関係のマッピングを含む、Temporal ワークフローとアクティビティの自動トレース。(2026-01-20, エージェント / RAG オブザーバビリティ)
- **レビュー用カンバンレイアウト**: フラグが立てられたスパンや人的レビューワークフローを管理するためのドラッグ＆ドロップインターフェース。(2026-01-20, 評価統合)
- **合理化されたオンラインスコアリング設定**: ログから直接、自動入力機能を使用してオンラインスコアリングルールを作成。(2026-01-20, 評価統合)
- **トレースページでの Loop**: 個別のトレースページで、要約やエラー分析のための AI アシスタント「Loop」が利用可能に。(2026-01-20, コア・オブザーバビリティ)

### [MLflow](https://mlflow.org/releases)
- **組織サポート**: 実験とリソースを整理するためのマルチワークスペース環境のサポート。(2026-02-12, エンタープライズ & セキュリティ)
- **MLflow アシスタント**: アプリやエージェントのデバッグを支援する Claude Code 搭載の製品内チャットボット。(2026-01-29, DevEx / 統合)
- **エージェント・パフォーマンス・ダッシュボード**: エージェントのレイテンシ、リクエスト数、品質スコアを監視するための構築済みチャート。(2026-01-29, モニタリング & メトリクス)
- **MemAlign Judge Optimizer**: 過去のフィードバックから評価ガイドラインを学習し、判定の精度を向上させるアルゴリズム。(2026-01-29, 評価統合)
- **Judge Builder UI**: カスタム LLM 判定プロンプトを作成、テスト、エクスポートするためのビジュアルインターフェース。(2026-01-29, 評価統合)
- **継続的オンラインモニタリング**: 本番環境の着信トレースに対して LLM 判定を自動的に実行。(2026-01-29, 評価統合)
- **分散トレース**: コンテキスト伝播を使用して、複数のサービスにわたるリクエストを追跡。(2026-01-29, コア・オブザーバビリティ)

### [Arize Phoenix](https://arize.com/docs/phoenix/release-notes)
- **OpenAI Responses API タイプのサポート**: Playground およびカスタムプロバイダーでの OpenAI API タイプ（Chat Completions vs Responses）の選択をサポート。(2026-02-12, DevEx / 統合)
- **データセット評価器**: 実験中にサーバー側で自動実行するために、評価器をデータセットに直接アタッチ。(2026-02-12, 評価統合)
- **Playground 用カスタムプロバイダー**: Playground とプロンプトで再利用可能な、カスタム AI プロバイダー（OpenAI、Azure、Anthropic など）の集中構成。(2026-02-11, DevEx / 統合)
- **Claude Opus 4.6 サポート**: 拡張思考パラメータとコスト追跡を備えた Anthropic の Claude Opus 4.6 モデルのサポート。(2026-02-09, コア・オブザーバビリティ)
- **ツール選択 & 呼び出し評価器**: エージェントのツール選択の正確性とパラメータ呼び出しの正しさを判定するための専用評価器。(2026-01-31, エージェント / RAG オブザーバビリティ)
- **設定可能なメール抽出**: JMESPath を使用した OAuth2 プロバイダーからのカスタムメール抽出のサポート。(2026-01-28, エンタープライズ & セキュリティ)
- **CLI コマンド**: ターミナルからプロンプト、データセット、実験を直接管理するための新しい CLI コマンド。(2026-01-22, DevEx / 統合)
- **スパンリンク付きトレースからデータセットへの変換**: ソーススパンへの双方向リンクを保持したまま、トレースをデータセットに変換。(2026-01-21, 評価統合)
- **アノテーションのエクスポート**: オフライン分析のためにトレースと共にアノテーションをエクスポートするための CLI サポート。(2026-01-19, DevEx / 統合)
- **CLI ターミナルアクセス**: AI コーディングアシスタントが CLI 経由で Phoenix データを照会可能に。(2026-01-17, DevEx / 統合)

### [W&B Weave](https://app.getbeamer.com/wandb/en)
- **オーディオモニター**: LLM 判定を使用して、テキストと共にオーディオ出力（MP3/WAV）を監視および判定するモニター。(2026-02-01, 評価統合)
- **ダイナミック・リーダーボード**: 永続的なカスタマイズと CSV エクスポート機能を備えた、評価から自動生成されるリーダーボード。(2026-01-29, 評価統合)
- **Playground でのカスタム LoRA**: Weave Playground でカスタム LoRA ウェイトを直接テストおよび比較するためのサポート。(2026-01-16, 実験 / 改善ループ)
