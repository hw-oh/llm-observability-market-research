# 週刊 LLM Observability 市場調査レポート
**日付**: 2026-02-12 | **モデル**: google/gemini-3-pro-preview | **データ収集日**: 2026-02-12

## 1. エグゼクティブ・サマリー

- MLflow は、ビジュアル Judge Builder と MemAlign オプティマイザーをリリースし、マルチワークスペース管理のための Organization Support を導入しました。
- LangSmith は、セルフホスト型プラットフォームをバージョン 0.13 にアップデートし、新しい CLI 機能やペアワイズ比較キューなどの開発者向けツールを拡張しました。
- Langfuse は、セキュリティ追跡用の Organization Audit Log Viewer をローンチし、セルフホスト版に PII マスキング機能を追加しました。
- Braintrust は、Cursor IDE サポートを通じて開発者ワークフローの統合を深化させ、Temporal および MCP サーバーアーキテクチャの Tracing を追加しました。
- Arize Phoenix は、OpenInference ベースの Tracing プラットフォーム内に、ツール選択と呼び出し精度のための専用 Eval を導入しました。
- W&B Weave は、マルチモーダル Eval のための高度なオーディオモニターと動的リーダーボードをデプロイし、専用および顧客管理のデプロイメントオプションをサポートしました。

**AI による市場インサイト**:

> MLflow によるビジュアル Judge Builder と MemAlign オプティマイザーの導入は、プログラムによる Eval とリーダーボードの可視化における Weave の差別化要因に挑戦するものです。
> Langfuse の専用 Audit Log Viewer と PII マスキングのリリースは、コンプライアンスの可視化に関する Weave の現在のエンタープライズ機能セットにおける特定のギャップを浮き彫りにしています。
> Braintrust の Cursor IDE や Temporal ワークフローとの直接的な統合は、複雑なエージェントのデバッグにおける主要な開発者中心プラットフォームとしての Weave のポジショニングを脅かしています。

## 3. 新機能 (直近 30 日間)

### [W&B Weave](https://app.getbeamer.com/wandb/en)
- **Audio Monitors**: LLM Judge を使用して、テキストと共にオーディオ出力 (MP3/WAV) を監視および判定するモニター。(2026-02-01, Evaluation Integration)
- **Dynamic Leaderboards**: Eval から自動生成されるリーダーボード。永続的なカスタマイズと CSV エクスポートが可能。(2026-01-29, Evaluation Integration)
- **Custom LoRAs in Playground**: Weave Playground で、カスタム Fine-tuning された LoRA の重みを直接ロードしてテストする機能。(2026-01-16, Experiment / Improvement Loop)

### [LangSmith](https://changelog.langchain.com)
- **Customize trace previews**: UI での Tracing プレビュー方法をカスタマイズする機能。(2026-02-06, Core Observability)
- **Google Gen AI Wrapper**: Google の Gen AI モデルに対する新しいラッパーサポート。(2026-01-31, DevEx / Integration)
- **LangSmith Self-Hosted v0.13**: エンタープライズデプロイメント向けのセルフホスト版アップデート。(2026-01-16, Enterprise & Security)

### [Langfuse](https://langfuse.com/changelog)
- **Run Experiments on Versioned Datasets**: 特定のバージョンのタイムスタンプでデータセットを取得し、再現性のために過去のバージョンで実験を実行。(2026-02-11, Experiment / Improvement Loop)
- **Org Audit Log Viewer**: セキュリティおよびアクセスイベントを追跡するための組織レベルの監査ログビューアー。(2026-02-09, Enterprise & Security)
- **Render Thinking/Reasoning**: Tracing 詳細ビューで、LLM レスポンスの思考 (Thinking) および推論 (Reasoning) 部分を個別にレンダリング。(2026-02-09, Core Observability)
- **Single Observation Evals**: 単一の Observation に対して直接 Eval を実行するサポート。(2026-02-09, Evaluation Integration)
- **Corrected Outputs for Traces**: Fine-tuning データセットを構築するために、Tracing ビューで LLM 出力の修正版を直接キャプチャ。(2026-01-14, Experiment / Improvement Loop)

### [Braintrust](https://braintrust.dev/docs/changelog)
- **Render attachments in custom views**: カスタム Tracing ビューで画像、動画、オーディオの直接レンダリングをサポート。(2026-02-01, Core Observability)
- **Navigate to trace origins**: ログ内の Tracing から、そのソースとなったプロンプトやデータセットの行へ移動。(2026-02-01, Core Observability)
- **Trace-level scorers**: カスタムコード Scoring が、マルチステップ Eval のために実行トレース全体にアクセス可能に。(2026-02-01, Evaluation Integration)
- **LangSmith integration**: Tracing および Eval の呼び出しを LangSmith と Braintrust の両方に送信するラッパー。(2026-02-01, DevEx / Integration)
- **Cursor integration**: Cursor エディタ内で Braintrust MCP サーバーを自動構成する拡張機能。(2026-02-01, DevEx / Integration)
- **Image rendering security controls**: ログ内の外部画像に対する構成可能なモード (自動ロード、クリックしてロード、ブロック)。(2026-02-01, Enterprise & Security)
- **Single span filters with aggregations**: 単一スパンフィルターと GROUP BY を組み合わせた集計 Tracing 分析。(2026-02-01, Monitoring & Metrics)
- **Auto-instrumentation for Python, Ruby, Go**: 主要言語に対するコード改変不要の Tracing サポート。(2026-01-29, DevEx / Integration)
- **Temporal integration**: Temporal ワークフローおよびアクティビティの自動 Tracing。(2026-01-21, DevEx / Integration)
- **Kanban layout for reviews**: フラグが立てられたスパンやレビューを管理するためのドラッグ＆ドロップインターフェース。(2026-01-21, Evaluation Integration)

### [MLflow](https://mlflow.org/releases)
- **Organization Support**: 実験やリソースを整理するためのマルチワークスペース環境のサポート。(2026-02-12, Enterprise & Security)
- **MLflow Assistant**: エージェントのデバッグや問題解決を支援する、Claude Code 搭載の製品内チャットボット。(2026-01-29, DevEx / Integration)
- **Agent Performance Dashboards**: レイテンシ、リクエスト数、品質 Scoring を監視するための構築済みチャート。(2026-01-29, Monitoring & Metrics)
- **MemAlign Judge Optimizer**: フィードバックから Eval ガイドラインを学習し、Judge の精度を向上させるアルゴリズム。(2026-01-29, Evaluation Integration)
- **Judge Builder UI**: カスタム LLM Judge プロンプトを作成、テスト、エクスポートするためのビジュアルインターフェース。(2026-01-29, Evaluation Integration)
- **Continuous Online Monitoring**: 本番環境の入力 Tracing に対して LLM Judge を自動的に実行。(2026-01-29, Monitoring & Metrics)
- **Distributed Tracing**: コンテキスト伝播を使用して、複数のサービスにわたるリクエストを追跡。(2026-01-29, Core Observability)

### [Arize Phoenix](https://arize.com/docs/phoenix/release-notes)
- **OpenAI Responses API Type Support**: Playground およびカスタムプロバイダーでの OpenAI API タイプ (Chat Completions vs Responses) の選択をサポート。(2026-02-12, DevEx / Integration)
- **Dataset Evaluators**: 実験中にサーバー側で自動実行するために、Eval をデータセットに直接アタッチ。(2026-02-12, Evaluation Integration)
- **Custom Providers for Playground**: Playground やプロンプトで再利用可能な、カスタム AI プロバイダー (OpenAI, Azure, Anthropic など) の集中構成。(2026-02-11, DevEx / Integration)
- **Claude Opus 4.6 Support**: 拡張された思考パラメータをサポートする Anthropic の Claude Opus 4.6 モデルのサポートを追加。(2026-02-09, DevEx / Integration)
- **Tool Selection & Invocation Evaluators**: エージェントが正しいツールを選択し、有効なパラメータで呼び出したかを評価する新しい Eval。(2026-01-31, Agent / RAG Observability)
- **Configurable Email Extraction**: JMESPath を使用した OAuth2 プロバイダーからのカスタムメール抽出のサポート。(2026-01-28, Enterprise & Security)
- **CLI Commands for Prompts/Datasets**: プロンプト、データセット、実験を管理するための CLI サポート (AI アシスタントへのパイプ渡しを含む)。(2026-01-22, DevEx / Integration)
- **Dataset Creation from Traces**: 双方向リンクのためにスパンの関連付けを保持したまま、本番 Tracing をデータセットに変換。(2026-01-21, Evaluation Integration)
- **Export Annotations with Traces**: 手動ラベルと Eval Scoring を含む Tracing をエクスポートするための CLI サポート。(2026-01-19, DevEx / Integration)
- **CLI Terminal Access for AI Assistants**: AI コーディングアシスタント (Cursor, Windsurf) が Phoenix データを照会できるように設計された CLI 機能。(2026-01-17, DevEx / Integration)

| Arize Phoenix | Tracing、Eval、データセットキュレーションを密接に統合する、開発者中心のオープンソース Observability プラットフォーム。 | エージェントワークフローや AI 支援開発環境 (CLI/IDE 統合) とのより深い統合へ移行。 | AI コーディングアシスタント向けの CLI ツール、専用のエージェントツール Eval、シームレスな Trace-to-Dataset ワークフローに重点を置いた最近のリリース。 |

## 5. エンタープライズ・シグナル

- Langfuse は、セキュリティとアクセスイベントを追跡するための Organization Audit Log Viewer をリリースしました。
- MLflow は、エンタープライズチーム向けにマルチワークスペース環境を管理するための Organization Support を導入しました。
- LangSmith は、セルフホスト版 (v0.13) をアップデートし、SOC 2/HIPAA コンプライアンスを維持しています。
- Braintrust は、画像レンダリングのセキュリティコントロールを追加し、オンプレミスデプロイメントをサポートしています。
- W&B Weave は、データの主権を確保するために、専用および顧客管理のデプロイメントオプションを提供しています。
