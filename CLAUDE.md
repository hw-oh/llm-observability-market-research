# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 프로젝트 개요

W&B Weave의 경쟁사 제품을 주기적으로 조사하여 기능 비교표(onepager)를 자동 생성하는 봇. 매주 웹 검색을 통해 경쟁사 정보를 수집하고, LLM으로 분석한 후 Markdown 파일로 출력한다. GitHub Pages를 통해 결과를 웹에서 열람할 수 있다.

## 주요 비교 대상

LangSmith, Arize Phoenix, Braintrust, LangFuse, Humanloop, Pydantic Logfire

## 비교 축

트레이싱/옵저버빌리티, 평가 파이프라인, 데이터셋 관리, 프롬프트 관리, 스코어링, LLM/프레임워크 통합, 가격, 셀프호스팅
